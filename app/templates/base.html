Sobre a plataforma

A GearHub é apenas uma plataforma de marketplace. Não solicitamos cadastro de cartão, planos ou assinaturas de nenhum tipo. Nossa plataforma não realiza a compra de nenhum item — caso se interesse por um produto listado em nosso catálogo, entre em contato diretamente com o vendedor através dos dados disponíveis na publicação.

Seus dados

Suas informações são únicas e pessoais, usadas apenas para verificação de usuário e confirmação de maioridade. Nunca solicitaremos seus documentos pessoais ou qualquer imagem sua.

Projeto acadêmico

Este é um projeto puramente acadêmico com o objetivo de aprender as bases fundamentais do Django num projeto fullstack C2C. Saiba mais em: github.com/ErickMazur29/Gear-Hub
